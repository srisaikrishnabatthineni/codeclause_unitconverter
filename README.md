# codeclause_unitconverter


Convert all kinds of standard units of measurement from one to another with this highly customizable.

It supports the following types of measurement by default.

 • Length

 • Weight

 • Speed

 • Temperature

 • Digital Storage
 
 Tools Used:

IDE: VS Code

Front-End

   • HTML

   • CSS

   • JavaScript
